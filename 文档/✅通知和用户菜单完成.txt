【功能说明】

1. 通知铃铛
   - 点击显示/隐藏通知列表
   - 有未读通知时显示红点
   - 显示3种类型通知:
     * 低库存提醒（警告）
     * 入库完成（信息）
     * 出库完成（成功）
   - 点击"全部已读"标记所有通知为已读

2. 用户头像
   - 显示用户名缩写（如: admin -> AD）
   - 点击显示用户菜单
   - 显示用户名和邮箱
   - 提供快捷操作链接
   - 退出登录功能

3. 交互体验
   - 点击按钮显示对应菜单
   - 打开一个菜单时自动关闭另一个
   - 点击页面其他地方自动关闭所有菜单
   - 平滑的过渡动画

───────────────────────────────────────────────────────────

【现在请执行】

1. 强制刷新浏览器
   Ctrl + Shift + R

2. 查看右上角
   - 应该看到铃铛图标（有红点）
   - 应该看到用户头像

3. 点击铃铛图标
   - 显示通知下拉菜单
   - 看到3条通知
   - 未读通知有蓝点标记

4. 点击"全部已读"
   - 所有通知变为已读状态
   - 红点消失

5. 点击用户头像
   - 显示用户菜单
   - 看到用户名和邮箱
   - 看到菜单选项

6. 点击"退出登录"
   - 弹出确认对话框
   - 确认后清除登录信息
   - 跳转到登录页

【通知数据结构】

当前使用模拟数据，将来可以对接真实API:

{
  id: 1,
  type: 'warning',      // warning/info/success/error
  title: '低库存提醒',
  message: '笔记本电脑包库存不足，当前库存: 5',
  time: '5分钟前',
  read: false
}

───────────────────────────────────────────────────────────

【用户信息来源】

从localStorage获取用户信息:
- localStorage.getItem('user_info')
- 包含: username, email等

如果已登录，会自动显示真实用户信息
如果未登录，显示默认信息

───────────────────────────────────────────────────────────

【退出登录流程】

1. 点击"退出登录"
2. 弹出确认对话框
3. 用户确认后:
   - 调用API.Auth.logout()
   - 清除localStorage中的token
   - 清除用户信息
   - 跳转到 /login/ 页面

───────────────────────────────────────────────────────────

【CSS样式】

使用Tailwind CSS类:
- rounded-apple: 圆角样式
- shadow-apple-hover: 悬浮阴影
- transition-colors: 颜色过渡
- hover:bg-light: 悬停背景
- z-50: 确保菜单在最上层
