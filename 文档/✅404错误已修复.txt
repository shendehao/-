【如果数据还是0】

可能的原因:

1. 后端API需要登录
   解决: 访问 http://localhost:8000/login/
   使用 admin/admin123 登录

2. 后端服务未启动
   解决: 运行 run.bat 启动服务器

3. 数据库没有数据
   解决: 运行 python manage.py shell
   然后执行:
   from apps.inventory.models import Item
   print(Item.objects.count())

  .then(data => {
    console.log('✅ API返回:', data);
    if (data.success) {
      console.log('总物品:', data.data.overview.total_items);
    }
  })
  .catch(err => console.error('❌ 错误:', err));
