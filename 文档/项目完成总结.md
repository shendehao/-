## 🚀 如何启动

### 方式一：一键启动（推荐）

```bash
# 双击运行
start.bat
```

### 方式二：Docker部署

```bash
docker-compose up -d
```

### 方式三：手动启动

```bash
# 1. 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 数据库迁移
python manage.py makemigrations
python manage.py migrate

# 4. 生成测试数据
python setup_database.py

# 5. 启动服务器
python manage.py runserver
```

## 📡 API端点总览

### 认证API
- `POST /api/auth/login/` - 用户登录
- `POST /api/auth/register/` - 用户注册
- `POST /api/auth/logout/` - 用户登出
- `POST /api/auth/token/refresh/` - 刷新Token
- `GET /api/auth/profile/` - 获取用户信息
- `PUT /api/auth/profile/` - 更新用户信息
- `POST /api/auth/change-password/` - 修改密码

### 库存物品API
- `GET /api/inventory/items/` - 获取物品列表
- `POST /api/inventory/items/` - 创建物品
- `GET /api/inventory/items/{id}/` - 获取物品详情
- `PUT /api/inventory/items/{id}/` - 更新物品
- `DELETE /api/inventory/items/{id}/` - 删除物品
- `GET /api/inventory/items/low_stock/` - 获取低库存物品
- `GET /api/inventory/items/statistics/` - 获取物品统计

### 类别API
- `GET /api/inventory/categories/` - 获取类别列表
- `POST /api/inventory/categories/` - 创建类别
- `GET /api/inventory/categories/{id}/` - 获取类别详情
- `PUT /api/inventory/categories/{id}/` - 更新类别
- `DELETE /api/inventory/categories/{id}/` - 删除类别

### 出入库操作API
- `GET /api/operations/` - 获取操作记录列表
- `POST /api/operations/` - 创建操作记录
- `GET /api/operations/{id}/` - 获取操作详情
- `POST /api/operations/inbound/` - 入库操作
- `POST /api/operations/outbound/` - 出库操作
- `GET /api/operations/statistics/` - 获取操作统计
- `GET /api/operations/recent/` - 获取最近操作

### 供应商API
- `GET /api/suppliers/` - 获取供应商列表
- `POST /api/suppliers/` - 创建供应商
- `GET /api/suppliers/{id}/` - 获取供应商详情
- `PUT /api/suppliers/{id}/` - 更新供应商
- `DELETE /api/suppliers/{id}/` - 删除供应商
- `GET /api/suppliers/active/` - 获取启用的供应商

### 仓库API
- `GET /api/warehouses/` - 获取仓库列表
- `POST /api/warehouses/` - 创建仓库
- `GET /api/warehouses/{id}/` - 获取仓库详情
- `PUT /api/warehouses/{id}/` - 更新仓库
- `DELETE /api/warehouses/{id}/` - 删除仓库
- `GET /api/warehouses/active/` - 获取启用的仓库

### 仪表盘API
